## 2. كل قناة بالتفصيل

### 2.1 📱 واتساب — القناة الذهبية

```
لماذا واتساب؟
├── 80%+ من اليمنيين يستخدمونه يومياً
├── يثقون في رسائله (مقارنة بـ SMS)
├── يدعم الوسائط (صور، PDF)
└── مجاني — مع Business API

خيارات واتساب:
┌────────────────────────────────────────────────────┐
│ الخيار        │ التكلفة     │ مميزات     │ عيوب     │
├────────────────┼─────────────┼────────────┼─────────┤
│ رقم شخصي عادي │ $0          │ مجاني      │ يحظر إذا │
│                │             │            │ أرسلت كثير│
│                │             │            │ (سبام)   │
├────────────────┼─────────────┼────────────┼─────────┤
│ WhatsApp Cloud │ $0-0.05/رسالة│ API رسمي  │ رسوم بعد │
│ API (Meta)     │             │ موثوق     | 1,000 رسالة│
│                │             │            │ مجاناً/شهر│
├────────────────┼─────────────┼────────────┼─────────┤
│ WATI           │ ~$50/شهر   │ سهل، دعم  │ غالي    │
│                │             │ عربي      │         │
├────────────────┼─────────────┼────────────┼─────────┤
│ Twilio WhatsApp│ $0.005/رسالة│ موثوق، API│ يحتاج 24س│
│                │             │ قوي       │ موافقة  │
└────────────────┴─────────────┴────────────┴─────────┘

التوصية:
├── 🥇 WhatsApp Cloud API (Meta): 1,000 رسالة مجانية/شهر
├── 🥈 WATI: إذا كبرنا (إدارة أفضل)
└── ❌ رقم شخصي: (للبداية فقط — مؤقت)
```

### 2.2 🔔 إشعار PWA (Progressive Web App)

```
كيف يعمل؟
├── المستخدم يزور الموقع
├── المتصفح يسأل: "هل تسمح بالإشعارات؟"
├── إذا وافق: نقدر نرسل له إشعار حتى لو الموقع مقفول
└── يدعم Chrome, Firefox, Safari (جوال)

التكلفة: $0 (مجاني — Push API)
الإعداد:
├── يحتاج Service Worker
├── يحتاج VAPID keys (توليد مجاني)
└── أسبوع تطوير

المميزات:
├── ✅ مجاني
├── ✅ فوري
├── ✅ لا يحتاج رقم هاتف
└── ✅ يعمل حتى لو البطارية قليلة (iOS)

العيوب:
├── ❌ المستخدم يجب أن يوافق أولاً
├── ❌ نسبة الموافقة في اليمن ~30-50%
├── ❌ محدود في iOS (Safari)
└── ❌ يتطلب اتصال إنترنت

التوصية:
├── 🟢 نفعّله — مجاني وأي إشعار إضافي مفيد
└── 🟡 لكن لا نعتمد عليه كلياً
```

### 2.3 📧 إيميل

```
المزودون:
┌───────────────────────────────────────────────┐
│ المزود     │ المجاني    │ السعر بعد    │ مميزات│
├─────────────┼────────────┼──────────────┼──────┤
│ SendGrid    │ 100/يوم    │ $19.95/50k   │ API  │
│ Sendinblue  │ 300/يوم    │ $25/40k      │ سهل  │
│ Gmail SMTP  │ 500/يوم    │ $0           │ مجاني│
│ SES (AWS)   │ 62k/شهر    │ $0.1/1k      │ رخيص│
└───────────────────────────────────────────────┘

المشكلة في اليمن:
├── 🔴 معظم اليمنيين لا يتفقدون الإيميل
├── 🔴 Gmail قد يضيف رسائلنا إلى Spam
└── 🟢 يستخدم فقط للمغتربين والموثوقين

التوصية:
├── 🟢 نضيفه كقناة ثانوية
├── 🟢 مجاني (SendGrid 100/يوم يكفي للبداية)
└── 🔴 لا نعتمد عليه في OTPs (بطيء)
```

```
المزودون:

├───────────────┼───────────────┼────────────────┤

└── 🟡 نستخدمه فقط في حالات الطوارئ
```

## 3. معمارية الإشعارات

### 3.1 الـ Fallback Chain (سلسلة الإحالات)

```
كل إشعار يمر على قنوات بالترتيب حتى يصل:

OTP (الأهم — أسرع وأضمن):
┌─────────────────────────────────────┐
│ 1️⃣ واتساب Cloud API                │
│    ↓ فشل (لا واتساب للرقم)           │
│ 2️⃣ PWA Push Notification           │
│    ↓ فشل (ما وافق على الإشعارات)     │
│ 3️⃣ SMS (احتياطي — مكلف)             │
│    ↓ فشل (الـ SMS ما وصل)            │
│ 4️⃣ ❌ فشل تام — معلّم للتصحيح اليدوي│
└─────────────────────────────────────┘

تحديث حالة (غير عاجل):
┌─────────────────────────────────────┐
│ 1️⃣ PWA Push Notification (مجاني)   │
│    ↓ ما في PWA                       │
│ 2️⃣ واتساب (غير عاجل — مرة واحدة)   │
│    ↓ فشل                             │
│ 3️⃣ إيميل                            │
│    ↓ فشل                             │
│ 4️⃣ ⏳ ينتظر — يُرسل مع الإشعار التالي│
└─────────────────────────────────────┘

تسويق (low priority):
┌─────────────────────────────────────┐
│ 1️⃣ إيميل (مجاني — SendGrid)        │
│    ↓ فشل                            │
│ 2️⃣ واتساب بث (مرة/شهر — بعناية)    │
│    ↓                                 │
│ 3️⃣ ✖️ لا نحاول أكثر (Spam)         │
└─────────────────────────────────────┘
```

### 3.2 الـ Queue Architecture

```
                   ┌──────────────┐
                   │  Notification│
                   │    Request    │
                   └──────┬───────┘
                          │
                   ┌──────▼───────┐
                   │    Queue     │
                   │   (Bull)     │
                   │   + Redis    │
                   └──────┬───────┘
                          │
                    ┌─────┴─────┐
                    │  Processor │
                    └─────┬─────┘
                          │
              ┌───────────┼───────────┐
              │           │           │
         ┌────▼────┐ ┌───▼───┐ ┌────▼────┐
         │ WhatsApp│ │ PWA   │ │ Email   │
         │ Queue   │ │ Queue │ │ Queue   │
         └─────────┘ └───────┘ └─────────┘
              │           │           │
         ┌────▼────┐      │      ┌────▼────┐
         │Meta API │      │      │SendGrid │
         └─────────┘      │      └─────────┘
                     ┌────▼────┐
                     │ Browser │
                     │ Push    │
                     └─────────┘

أولويات Queue:
├── 🔴 Priority 1: OTP (فوري)
├── 🟡 Priority 2: تأكيدات (> 1 دقيقة)
├── 🟢 Priority 3: تحديثات حالة (> 5 دقائق)
└── ⚪ Priority 4: تسويق (> ساعة)
```
