# التصاميم التفصيلية - Detailed UI States

## Qor3a — كل شاشة بحالاتها الكاملة

---

## 1. المبدأ: كل شاشة لها 4 حالات

| الحالة | الوصف | يظهر عندما |
|--------|-------|-----------|
| 🟢 **Default** | تعمل بشكل طبيعي | كل شيء متاح |
| ⏳ **Loading** | تنتظر بيانات | جلب من API |
| 🚫 **Empty** | لا توجد بيانات | قائمة فاضية |
| ❌ **Error** | حدث خطأ | فشل API |
| ✅ **Success** | تمت العملية (للإجراءات) | نجاح عملية |
