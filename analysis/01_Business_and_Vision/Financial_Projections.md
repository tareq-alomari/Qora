```

├── 90% من العملاء يدفعون 1,000 YR
├── 10% يدفعون 800 YR (أول 100)
└── المتوسط المرجح: 980 YR

إيرادات متوقعة (سيناريو متوقع):
├── السنة 1: 1,000 × 980 = 980,000 YR
├── السنة 2: 2,000 × 1,000 = 2,000,000 YR
└── السنة 3: 3,500 × 1,200 (بعد زيادة) = 4,200,000 YR
```

## 3. التدفق النقدي (Cash Flow)

### 3.1 لماذا التدفق النقدي أهم من الأرباح؟

```
المشكلة:
├── الفلوس تدخل في سبتمبر-أكتوبر فقط (موسم)
├── الفلوس تطلع كل شهر (رواتب، سيرفر...)
└── لازم نضمن الفلوس تكفي الـ 12 شهر

الحل:
├── ندّخر من موسم التسجيل لنغطي باقي السنة
├── أو نحتاج رأس مال إضافي لتغطية الشهور العجاف
└── نعرف بالضبط كم نحتاج في البنك
```

## 4. تحليل Break-Even بالشهور

```
Break-Even Point شهرياً:
├── التكاليف الثابتة الشهرية: ~110,000 YR (في غير الموسم)
├── التكاليف المتغيرة لكل طلب: ~200 YR
├── سعر البيع: 1,000 YR
└── الربح لكل طلب: 800 YR

في غير الموسم:
├── BE = 110,000 ÷ 800 = 138 طلب/شهر
├── هذا صعب جداً خارج الموسم
└── ← نعتمد على ادخار الموسم

في الموسم:
├── التكاليف الشهرية: ~180,000 YR (زيادة تسويق + دوام أطول)
├── BE = 180,000 ÷ 800 = 225 طلب/شهر
└── 225 طلب/شهر = ~8 طلبات/يوم ← سهل

كم طلب نحتاج لتعويض سنة كاملة؟
├── التكاليف السنوية: 1,490,000 YR
├── الربح لكل طلب: 800 YR
├── 1,490,000 ÷ 800 = 1,863 طلب/سنة
└── في موسمين = ~932 طلب/موسم
```

## 5. تحليل الحساسية (Sensitivity Analysis)

### 5.1 ماذا لو تغير السعر؟

```
السعر     │ الربح لكل طلب │ BE (طلبات/سنة) │ BE (موسم واحد)
──────────┼───────────────┼──────────────────┼─────────────────
800  YR   │    600 YR     │    2,483         │    1,242
1,000 YR  │    800 YR     │    1,863         │      932
1,200 YR  │  1,000 YR     │    1,490         │      745
1,500 YR  │  1,300 YR     │    1,146         │      573
```

### 5.2 ماذا لو زادت التكاليف؟

```
التكاليف الثابتة      │ BE (طلبات/سنة)  
──────────────────────┼─────────────────
1,490,000 YR (أساسي)  │    1,863
+20% = 1,788,000 YR   │    2,235
+50% = 2,235,000 YR   │    2,794
-20% = 1,192,000 YR   │    1,490
```

### 5.3 متى نرفع السعر؟

```
شروط رفع السعر:
├── 🟢 بعد الموسم الأول (لدينا عملاء)
├── 🟢 إذا زادت التكاليف (تضخم، رواتب)
├── 🟢 إذا أضفنا خدمات جديدة (باقات)
└── 🟢 إذا كان المنافسون أغلى (2,000 YR)

لا نرفع السعر:
├── 🔴 قبل أن نبني قاعدة عملاء
├── 🔴 إذا كان المنافسون ينخفضون
└── 🔴 في أول موسم (نحن جدد)
```
