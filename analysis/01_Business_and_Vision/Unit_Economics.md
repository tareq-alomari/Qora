## 2. هامش الربح (Profit Margin Analysis)

### 2.1 لكل طلب

```
سعر البيع:          1,000 YR
التكلفة المباشرة:   ~200 YR
─────────────────────────────
هامش الربح الخام:    800 YR  (80%)
التكلفة غير المباشرة: ~50 YR (مقسومة على عدد الطلبات)
─────────────────────────────
صافي الربح:         ~750 YR  (75%)
```

### 2.2 Break-Even Analysis

```
التكاليف الثابتة السنوية: 1,675,000 YR
هامش الربح لكل طلب:           800 YR

Break-Even Point: 1,675,000 ÷ 800 = 2,094 طلب/سنة
                    = ~175 طلب/شهر
                    = ~6 طلبات/يوم

هل هذا ممكن؟
├── 🟢 في الموسم (سبتمبر-أكتوبر):
│   175 طلب/شهر = 6 طلبات/يوم → سهل جداً
│
├── 🟡 خارج الموسم:
│   175 طلب/شهر صعب → نعتمد على خدمات أخرى
│
└── 💡 الحل: الموسم فقط ← نغطي التكاليف كلها
```

### 2.3 سيناريوهات الإيرادات

```
سيناريو متحفظ (300 طلب/موسم):
├── الإيرادات: 300 × 1,000 = 300,000 YR
├── التكاليف المباشرة: 300 × 200 = 60,000 YR
├── التكاليف الثابتة: 1,675,000 YR
└── الخسارة: -1,435,000 YR ❌

سيناريو متوسط (500 طلب/موسم):
├── الإيرادات: 500 × 1,000 = 500,000 YR
├── التكاليف المباشرة: 500 × 200 = 100,000 YR
├── التكاليف الثابتة: 1,675,000 YR
└── الخسارة: -1,275,000 YR ❌

سيناريو جيد (2,000 طلب/موسم):
├── الإيرادات: 2,000 × 1,000 = 2,000,000 YR
├── التكاليف المباشرة: 2,000 × 200 = 400,000 YR
├── التكاليف الثابتة: 1,675,000 YR
└── الخسارة: -75,000 YR ❌ (تقريباً Break-even)

سيناريو ممتاز (3,000 طلب/موسم):
├── الإيرادات: 3,000 × 1,000 = 3,000,000 YR
├── التكاليف المباشرة: 3,000 × 200 = 600,000 YR
├── التكاليف الثابتة: 1,675,000 YR
└── الربح: 725,000 YR ✅
```

## 4. استراتيجية التسعير (Pricing Strategy)

### 4.1 مقارنة السيناريوهات

| السيناريو | السعر | العملاء المطلوبون للـ BE | الربح عند 2,000 عميل | 
|-----------|-------|-------------------------|---------------------|
| فقط لوتري | 1,000 YR | 2,094 | -75,000 YR |
| فقط لوتري | 1,500 YR | 1,290 | 1,325,000 YR |
| لوتري + فحص نتيجة | 1,500 YR | 1,290 | 1,325,000 YR |
| لوتري + فيزا + جواز | 1,000 (لوتري) + خدمات أخرى | 700 (إجمالي) | 2,500,000+ YR |

### 4.2 التوصية

```
🏆 الخطة الموصى بها:

1️⃣ DV Lottery: 1,000 YR (مغناطيس — يجيب العملاء)
2️⃣ فحص النتيجة: مجاناً (مكافأة للعميل)
3️⃣ خدمات إضافية (لاحقاً): فيزا، جواز، ترجمة ← 2,000-5,000 YR
4️⃣ باقات: لوتري + فحص النتيجة + استشارات = 1,500 YR

الاشتراك السنوي (لاحقاً):
├── 🥇 ذهبي: 3,000 YR — لوتري + فحص نتيجة + استشارات
├── 🥈 فضي: 2,000 YR — لوتري + فحص نتيجة
└── 🥉 برونزي: 1,000 YR — لوتري فقط
```
