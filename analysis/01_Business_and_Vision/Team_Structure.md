## 2. المسمى الوظيفي لكل دور

### 2.1 المؤسس/المدير (Founder/CEO)

```
المسمى: المؤسس والمدير التنفيذي
الدوام: بدوام كامل (في البداية)
الراتب: 0 (يأخذ من الأرباح)

المسؤوليات:
├── ✅ الرؤية والاستراتيجية
├── ✅ العلاقات مع الموردين (كريمي، جيب...)
├── ✅ التسويق (في البداية)
├── ✅ العلاقات مع العملاء الكبار
├── ✅ القرارات المالية
└── ✅ متابعة التنفيذ

مؤهلات:
├── فهم بسوق اللوتري والخدمات
├── خبرة في إدارة المشاريع
├── شبكة علاقات في اليمن
└── رغبة في العمل لساعات طويلة
```

### 2.2 المطور Fullstack (Developer)

```
المسمى: مطور Fullstack (Node.js + React)
الدوام: جزئي (20-30 ساعة/أسبوع) — قابل للتحول لكامل
الراتب: $500-800/شهر (400,000-650,000 YR) حسب الخبرة
المكان: عن بُعد (يفضّل يمني)

المسؤوليات:
├── ✅ تطوير الـ API (Node.js + Express)
├── ✅ تطوير الواجهات (React + Tailwind)
├── ✅ إدارة قاعدة البيانات (PostgreSQL)
├── ✅ CI/CD والنشر
├── ✅ صيانة وحل المشاكل التقنية
└── ✅ AI Service integration

مؤهلات:
├── خبرة 2+ سنوات في Node.js
├── React
├── PostgreSQL
├── Docker
└── فهم بالـ State Machine

الجدول الزمني:
├── موسم التسجيل (سبتمبر-أكتوبر): دوام كامل
├── ماي (نتائج): دوام كامل
└── باقي السنة: دوام جزئي (صيانة + تحسينات)
```

### 2.3 موظف التشغيل (Operations)

```
المسمى: موظف تشغيل / Customer Success
الدوام: بدوام كامل
الراتب: $300-400/شهر (240,000-320,000 YR)
المكان: يفضل حضوري (صنعاء أو عدن)

المسؤوليات:
├── ✅ مراجعة بيانات العملاء
├── ✅ تأكيد الإيصالات (Payment Verification)
├── ✅ التواصل مع العملاء على واتساب
├── ✅ الإدخال الرسمي في موقع وزارة الخارجية
├── ✅ متابعة الطلبات والتأكد من اكتمالها
└── ✅ حل مشاكل العملاء (L2 support)

مؤهلات:
├── 📱 يجيد استخدام الكمبيوتر والإنترنت
├── 📱 صبور في التعامل مع العملاء
├── 📱 أمانة (يتعامل مع بيانات حساسة)
└── 📱 لغة إنجليزية مقبولة (قراءة النموذج الرسمي)

الجدول الزمني:
├── موسم التسجيل: 8 صباحاً - 8 مساءً (12 ساعة) 6 أيام
├── ماي (نتائج): 10 صباحاً - 6 مساءً
└── باقي السنة: 9 صباحاً - 3 مساءً (أو عطلة)
```

### 2.4 مصمم UI/UX (اختياري — لعمل التصاميم)

```
المسمى: مصمم UI/UX
الدوام: جزئي (10-15 ساعة/شهر)
الراتب: $200-400/شهر (160,000-320,000 YR) حسب المشروع
المكان: عن بُعد

المسؤوليات:
├── ✅ تصميم الشعار والهوية البصرية
├── ✅ تصميم الصفحات الجديدة
├── ✅ تحسين تجربة المستخدم
└── ✅ مواد تسويقية (صور الإعلانات)

مؤهلات:
├── خبرة في Figma
├── فهم بالـ UI/UX
└── إحساس جمالي جيد
```
