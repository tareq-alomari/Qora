## 1. دورة حياة العميل (Customer Lifecycle)

### 1.1 التقويم السنوي الكامل

```
  ┌─────────────────────────────────────────────────────────────────┐
  │               دورة حياة عميل Qor3a (سنة كاملة)                    │
  └─────────────────────────────────────────────────────────────────┘

شهر 9-10 (سبتمبر-أكتوبر): 📝 موسم التسجيل
  ┌── Acquisition (جذب العملاء)
  │   ├── التسويق: تيك توك، واتساب، تيليجرام
  │   ├── التسجيل: العميل يملأ الطلب
  │   └── الدفع: يحول ويؤكد
  │
  │   عميل جديد 🆕 → 🟢 نشط

شهر 11-4 (نوفمبر-أبريل): ⏳ انتظار
  ┌── Engagement (ابق على تواصل)
  │   ├── إشعارات: "طلبك قيد المعالجة"
  │   └── محتوى: نصائح هجرة، استعداد للتأشيرة
  │
  │   🟢 نشط → 🟡 خامل (مخاطرة)

شهر 5 (مايو): 🎯 موسم النتائج
  ┌── Activation (إعادة نشاط)
  │   ├── "النتائج قرب تظهر!"
  │   ├── فحص النتيجة تلقائياً
  │   └── إشعار الفائزين/الخاسرين
  │
  │   🟡 خامل → 🔴 فائز 🥳 / 🔵 خاسر

شهر 6-8 (يونيو-أغسطس): 🛑 خارج الموسم
  ┌── Retention (ابق العميل)
  │   ├── خدمات جديدة (فيزا، جواز، ترجمة)
  │   ├── خصم الموسم القادم
  │   └── منشورات توعوية
  │
  │   🔴 فائز → 🟣 VIP (مستعد للسفر)
  │   🔵 خاسر → 🟢 عميل محتمل للموسم الجديد
```

## 2. استراتيجيات الاحتفاظ (Retention Strategies)

### 2.1 الفائزين (Winners) — ~1-3% من العملاء

```
الفائز يحتاج فجأة:
├── ✈️ استشارات سفر (تأشيرة، تذاكر)
├── 📄 ترجمة المستندات (شهادة زواج، شهادات جامعية)
├── 🏥 تأمين صحي
├── 💵 تحويل أموال
└── 🏠 استشارات إسكان في أمريكا

نحن نقدم:
├── ✅ استشارة مجانية لمدة 30 دقيقة
├── ✅ خصم 20% على ترجمة المستندات
├── ✅ دليل "أول 30 يوم في أمريكا"
└── ✅ متابعة حتى الوصول

قيمة العميل الفائز = 5,000-20,000 YR (ترجمة + استشارات)
```

### 2.2 الخاسرين (Losers) — ~97-99% من العملاء

```
الخاسر يشعر بخيبة أمل ويحتاج:
├── 🎯 رسالة دعم: "نحن معك، جرب السنة القادمة"
├── 🎁 خصم الموسم القادم: 15% خصم لـ 3 أيام
├── 📝 تذكير تلقائي: "موسم التسجيل القادم"
└── 🆕 خدمات بديلة: فيزا سياحية؟ دراسة؟

رسالة للخاسر:
══════════════════════════
نعم، لم توفق هذه المرة.
لكن آلاف اليمنيين يسجلون كل سنة لسنوات حتى يفوزوا.
السنة القادمة حاول مرة أخرى معانا — بخصم 15%.
══════════════════════════

نسبة العودة للموسم القادم: ~30-40%
```

### 2.3 الحائدين (Churned — لم يسجل مرة أخرى)

```
لماذا رحل؟
├── 💸 غالي؟ ← نقدم خصم
├── 😠 تجربة سيئة؟ ← نعتذر ونتحسن
├� 🤷 ما فكر في اللوتري السنة الجديدة؟ ← نذكره
└── 🏢 راح لمكتب آخر؟ ← نتفهم (نسأله ليه؟)

Win-back Campaign (حملة استعادة):
├── إيميل/واتساب: "اشتقنا لك! خصم 25% للتسجيل المبكر"
├── تذكير: "باقي 30 يوم على إغلاق التسجيل"
├── مجاني: "فحص صورتك مجاناً هذه السنة"
└── استبيان: "كيف نقدر نخدمك أفضل؟"

نسبة الاستعادة: ~10-20%
```

## 3. قنوات التواصل لكل مرحلة

| المرحلة | القناة | التكرار | المحتوى |
|---------|--------|---------|---------|
| 🆕 **بعد التسجيل** | واتساب | مرة واحدة | "تم استلام طلبك بنجاح ✅" |
| ⏳ **في الانتظار** | إيميل | شهرياً | "طلبك قيد المعالجة + نصائح" |
| 🎯 **قبل النتائج** | واتساب | أسبوعياً (مايو) | "النتائج على وشك الظهور!" |
| 🥳 **فائز** | اتصال + واتساب | فوري | مبروك! + الخطوات القادمة |
| 😔 **خاسر** | واتساب | مرة واحدة | رسالة دعم + خصم للموسم القادم |
| 🛑 **خارج الموسم** | إيميل/تيليجرام | شهرياً | محتوى مفيد + خدمات بديلة |
| 🤝 **توصية** | واتساب | مرة واحدة | "ادعي صديقك وخذ خصم" |

## 4. مقاييس الاحتفاظ (Retention Metrics)

| المقياس | التعريف | الهدف |
|---------|---------|-------|
| 🟢 **Churn Rate** | % العملاء الذين لم يعودوا | < 70% |
| 🔄 **Repeat Rate** | % العملاء الذين سجلوا مرة ثانية | > 25% |
| 📢 **Referral Rate** | % العملاء الذين جلبوا صديق | > 10% |
| 😊 **NPS (Net Promoter Score)** | كم عميل يوصي بنا | > 50 |
| 💬 **Engagement Rate** | % العملاء الذين يفتحون إشعاراتنا | > 40% |
| 📅 **Time to Re-register** | كم يوم يعود العميل للتسجيل | < 365 يوم |

## 5. استراتيجية التوسع (خدمات إضافية)

### 5.1 خدمات يمكن إضافتها بعد اللوتري

```
خدمات هجرة:
├── 📋 استشارات الهجرة إلى أمريكا
├── 📄 ترجمة المستندات (شهادات، وثائق)
├── ✈️ حجوزات سفر إلى أمريكا
└── 🏠 استشارات سكن للفائزين

خدمات حكومية (إلكترونية):
├── 🛂 تجديد جواز السفر
├── 📝 تسجيل فيزا سياحية
├── 🏛️ توثيق عقود
└── 🎓 معادلة الشهادات

خدمات مالية:
├── 💵 تحويل أموال (حوالات)
├── 📱 شحن رصيد جوال
└── 🏦 فتح حساب بنكي (في أمريكا)
```
