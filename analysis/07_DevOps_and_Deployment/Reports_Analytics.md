## 1. أنواع التقارير

### 1.1 تقارير تشغيلية (يومية/أسبوعية)

```
للتشغيل اليومي — من يحتاجها: الموظف + المدير
الهدف: مراقبة سير العمل
التكرار: يومي (صباحاً)
```

### 1.2 تقارير إدارية (شهرية/سنوية)

```
للقرارات — من يحتاجها: المدير
الهدف: تحليل الأداء، اتخاذ قرارات
التكرار: شهري + نهاية الموسم
```

### 1.3 تقارير مالية (أسبوعية/شهرية)

```
للإيرادات — من يحتاجها: المدير + المحاسب
الهدف: متابعة الفلوس
التكرار: أسبوعي
```


### 4.2 وظيفة SQL للتقرير اليومي

```sql
-- تقرير يومي: ملخص سريع
SELECT 
  COUNT(*) AS total_orders,
  COUNT(*) FILTER (WHERE status = 'payment_verification') AS pending_payment,
  COUNT(*) FILTER (WHERE status = 'completed') AS completed,
  COUNT(*) FILTER (WHERE created_at::date = CURRENT_DATE) AS today_orders,
  COALESCE(SUM(amount) FILTER (WHERE created_at::date = CURRENT_DATE), 0) AS today_revenue
FROM orders
WHERE created_at > NOW() - INTERVAL '7 days';
```
