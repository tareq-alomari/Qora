📡 نقاط النهاية API

  • GET /health — فحص صحة الخدمة
  • POST /api/v1/auth/register — تسجيل مستخدم جديد
  • POST /api/v1/auth/verify-otp — تأكيد رمز التحقق
  • POST /api/v1/auth/login — طلب رمز دخول
  • POST /api/v1/auth/refresh — تجديد التوكن
  • GET /api/v1/users/profile — الملف الشخصي
  • PATCH /api/v1/users/profile — تحديث الملف الشخصي
  • POST /api/v1/orders — إنشاء طلب جديد
  • GET /api/v1/orders — قائمة الطلبات
  • POST /api/v1/orders/{id}/photo — رفع الصورة
  • POST /api/v1/orders/{id}/payment/receipt — رفع إيصال الدفع
  • PATCH /api/v1/orders/{id}/status — تغيير حالة الطلب
  • GET /api/v1/payments/methods — طرق الدفع المتاحة
  • GET /api/v1/admin/stats — إحصائيات لوحة التحكم
  • GET /api/v1/admin/settings — إعدادات النظام
